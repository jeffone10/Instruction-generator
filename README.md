## üìå Project Objective
The objective of this project is to build an **automated tool that generates step-by-step user manual guides for any website** by converting them into clear, professional documentation. This tool helps users create manuals without the usual manual effort by automatically capturing actions, understanding the user interface, and producing downloadable documentation.

## üß† Problem Statement
Creating user manuals is traditionally:
- Time-consuming
- Error-prone
- Difficult to maintain with UI changes

These challenges result in delays and inconsistencies in user guidance.

## üöÄ Solution Overview
The Instruction-generator works by:

1. Capturing user interactions on a website, including clicks, navigation, and inputs.
2. Capturing relevant UI screenshots during these interactions.
3. Extracting meaningful context from the user interface.
4. Converting raw interaction logs into structured, readable steps.
5. Generating a downloadable **PDF user manual** that you can easily share with users.

This streamlined process simplifies how companies document their websites.
